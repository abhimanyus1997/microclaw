‚ö° Quick Start

Follow these steps to get your agent running in minutes.

1. Installation

Install dependencies using uv (recommended) or pip.

uv pip install -r tools/requirements.txt

2. Setup & Flash

Launch the manager to configure credentials and flash the firmware.

python tools/microclaw.py setup

3. Launch Dashboard

Start the local manufacturing dashboard.

python main.py

Open http://localhost:8000 to chat with your agent.

üß† AI Capabilities

MicroClaw is "Self-Aware" thanks to Function Calling tools defined in the firmware.

Available Tools

  • claw_control({action: "OPEN"|"CLOSE"|"WAVE"}) - Move the servo.
  • gpio_control({pin: int, mode: "output", state: 0|1}) - Control any pin.
  • get_system_stats() - Read Free Heap, CPU Freq, Flash Size.
  • memory_write({content: "..."}) - Save facts to permanent storage.

Example Prompt Interactions

"Open the claw!" ‚Üí Calls claw_control(OPEN).

"How much memory do you have?" ‚Üí Calls get_system_stats().

"Turn on the light on pin 4" ‚Üí Calls gpio_control(4, output, 1).

üì° API Reference

Control MicroClaw programmatically via the microclaw.py server.

POST /api/chat

Send a message to the agent.

{
  "text": "Hello Microclaw!",
  "provider": "groq" // optional
}

POST /api/flash

Upload and flash a new firmware binary.

curl -X POST -F "file=@firmware.bin" -F "port=/dev/ttyUSB0" \
  http://localhost:8000/api/flash
